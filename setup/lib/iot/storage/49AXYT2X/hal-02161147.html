Abstract : Industry 4.0 opens new opportunities and challenges. One of these challenges is to provide a set of open and standardized technologies. The long life IT technologies such Ethernet and Wi-Fi can be considered as potential candidate. In this article, we present the industry 4.0 and related technologies like historical and candidate network technologies for the industry , IIoT and cognitive networks. We propose a concept using cognitive networks combined with machine learning to make IT networking technologies compliant with industrial requirements.
